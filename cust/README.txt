-----------TO INSTALL THIS CUSTOM FIX IN CLIENT TOOLS-----------
1 - Shut down Spoon, Kitchen, Pan, or Carte
2 - Extract this zip file into the data-integration directory 
   (this will add folders under data-integration/classes)
3 - Move the original "excel-plugins" plugin folder out of data-integration/plugins to a backed up location
4 - Extract "excel-plugins-9.5.0.0-240-CSS-CUST-PDI-19805.zip" into data-integration/plugins
5 - Restart the tool


-----------TO INSTALL THIS CUSTOM FIX IN PENTAHO SERVER-----------
1 - Shut down the Pentaho Server
2 - Extract this zip file into the pentaho-server/tomcat/webapps/pentaho/WEB-INF directory
    (this will add folders under pentaho-server/tomcat/webapps/pentaho/WEB-INF/classes)
3 - Move the original "excel-plugins" plugin folder out of pentaho-solutions/system/kettle/plugins to a backed up location
4 - Extract "excel-plugins-9.5.0.0-240-CSS-CUST-PDI-19805.zip" into pentaho-solutions/system/kettle/plugins
5 - Restart the Pentaho Server
